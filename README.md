
## 概述
* 例子来自 《node.js 实战》这本书的第一章。
* 用 coffeeScript & express4 & bootstrap 重构。
* 在这编码过程以及部署过程中遇到很多坑，希望这个例子能够帮到同样遇坑的朋友。

## 功能
* 多人注册、登录
* 发布文章 `(markdown语法)`
* 发布评论 `(markdown语法)`
* 上传文件
* 用户文章页
* 文章的编辑删除
* 存档浏览
* 标签浏览
* 分页功能
* 文章pv `（浏览数）`
* 文章搜索
* 系统日志记录
