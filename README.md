
## 概述
* 例子来自 《node.js 实战》这本书的第一章。
* 用 coffeeScript & express4 & bootstrap 重构。
* 在这编码过程以及部署过程中遇到很多坑，希望这个例子能够帮到同样遇坑的朋友。

## 功能
1. 多人注册、登录
2. 发布文章 `(markdown语法)`
3. 发布评论 `(markdown语法)`
4. 上传文件
5. 用户文章页
6. 文章的编辑删除
7. 存档浏览
8. 标签浏览
9. 分页功能
10. 文章pv `（浏览数）`
11. 文章搜索
12. 系统日志记录
